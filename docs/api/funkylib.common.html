Exceptional<A>

Either<Exception, A> analogue. Either Represents a computation that may either result in an exception, or return a successfully computed value. 

try {
  return IO.@return(() => new Exceptional<StreamReader>(System.IO.File.OpenText(path)));
}
catch (Exception ex) {
  return IO.@return(() => new Exceptional<StreamReader>(ex));
}
