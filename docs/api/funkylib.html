Either

Represents a value of one of two possible types (a disjoint union). An instance of `Either` holds Left<L>(L) or Right<R>(R).

A common use of `Either` is as an alternative to Option for dealing with possibly missing values. In this usage, None is replaced with a Left<L>(L) which can contain useful information. Right<R>(R) takes the place of Some<A>(A). Convention dictates that `Left` is used for failure and `Right` is used for success.
